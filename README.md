# Algoritmos em C# - Grokking Algorithms

Este repositório contém minhas implementações e resoluções de exercícios baseadas no livro **"Entendendo Algoritmos" (Grokking Algorithms)** de Aditya Bhargava.

O foco deste projeto é fortalecer os fundamentos de Ciência da Computação, explorando lógica de programação, estruturas de dados e análise de complexidade (Big O).

## Tecnologias Utilizadas
* **Linguagem:** C#
* **Plataforma:** .NET
* **IDE:** Visual Studio
---

## Conteúdo Estudado

### Capítulo 4: Recursividade
Nesta etapa, explorei o conceito de **Dividir para Conquistar (D&C)** e como a **Pilha de Chamadas (Call Stack)** funciona na memória.

* **Exercício 4.1:** Implementação de uma função recursiva para somar os itens de um array.
* **Exercício 4.2:** Função recursiva para contar o número de itens em uma lista dinâmica (`List<int>`).
* **Exercício 4.3:** Algoritmo recursivo para encontrar o maior valor em uma lista (Busca Linear Recursiva).

## Aprendizados Chave
* **Caso Base vs. Caso Recursivo:** A importância de definir uma condição de parada para evitar o `StackOverflowException`.
* **Gerenciamento de Estado:** Como passar acumuladores entre chamadas de função.
